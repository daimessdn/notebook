# membuat page

1. untuk membuat route "/", kita bekerja di file `index.js`

   ```react
   function Home() {
       return (
           <h1>The Home Page</h1>
       )
   }
   
   export default Home;
   ```

2. untuk membuat route "/news", kita bekerja di file `news.js`

   ```react
   function News() {
       return (
           <h1>The News Page</h1>
       )
   }
   
   export default News;
   ```
