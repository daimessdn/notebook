# instalasi

1. buat environment variable project dan aktifkan environment tersebut.

   ```bash
   virtualenv venv
   source venv/bin/activate
   
   # atau
   python3 -m venv venv
   source venv/bin/activate
   ```

2. install **flask**

   ```bash
   pip install flask
   ```

3. install python-venv

   ```bash
   pip install python-dotenv
   ```
